# Ucollege
![Logo de Ucollege](/img/UcollegeLogo.png)
### Descripción Breve:

Ucollege es un software educativo diseñado para facilitar la matrícula de estudiantes en línea y su integración en un campus virtual. El sistema también incluye la inscripción para las pruebas de aptitud necesarias para el ingreso a la institución.

### Tecnologías Utilizadas:
Node.js, React, Node Express, PostgreSQL, Prisma.

### Funcionalidades Clave:

Gestión integral de estudiantes.
Proceso de matrícula eficiente.
Administración de exámenes de admisión.
Campus virtual para la distribución de información y clases.
Proceso de Desarrollo:
El cliente solicitó el sistema, y a través de una encuesta abierta, se recolectaron requisitos funcionales y no funcionales. Se definió una arquitectura utilizando API RESTful, y el equipo se dividió en frontend y backend. Se utilizó Scrum para manejar los tiempos ajustados.

### Desafíos Enfrentados:
Los principales desafíos incluyeron la gestión del tiempo debido a plazos ajustados y la implementación de tecnologías nuevas como Node Express. Los cambios constantes en los requisitos llevaron a ajustes en el diseño de la base de datos, que se mitigaron normalizando las tablas y simplificando la estructura.

### Lecciones Aprendidas:
Adquirí habilidades en nuevas tecnologías, aprendí a trabajar bajo presión y a utilizar la metodología Scrum. También mejoré en la administración del tiempo y en el uso de herramientas como Git y Prisma.
